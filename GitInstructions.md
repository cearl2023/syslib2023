# GIT Instructions

## Open Directory
- cd
  - repos/syslib2023/

## Creat New Document for Journal in Nano

- Create new document

## Push to Github
- git status
- git add "journalweekname.md"
- git commit -m "journalweekname.md"
- git push origin main
